教学效能辅助决策系统 v2.0（课堂人工智能系统）
2026年计算机设计竞赛参赛作品

一个实时课堂状态分析平台，集成了深度学习算法和现代教育管理理念。

📖 项目深度背景
在当前智能教育的背景下，传统的“监控”出勤已不再符合以人为本教学的趋势。 该项目由一组人工智能专业人士开发，负责人同时担任班级委员会成员，理解课堂互动中的痛点。 该系统不再只是“检测工具”，而是一个教学观察台，旨在通过非侵入式视觉分析为教师提供即时的教学质量反馈。

核心理念
温和的感知：界面设计摒弃了冷峻的工业感，采用了极简主义视觉语言。

辅助决策：通过分析专注趋势（0-100分），帮助教师判断何时深入解释，何时调整互动。

多终端连接：提供高性能的 PyQt6 桌面和跨平台 Streamlit 网页终端。

✨ 核心功能列表
高精度专注建模：基于MediaPipe和OpenCV，实时捕捉面部方向、眼神焦点和坐姿特征，并将其转化为定量焦点评分。

实时表现波动图表：将专业坐标轴锁定在0到105之间，以避免数据跳跃，并直观地显示课堂的“黄金时间”。

拟人观察体验：内置决策引擎每5-10秒生成温和的教学建议（例如，“在校学生状态极佳，建议输入核心知识点进行解释”）。

数据持久性和追溯：自动记录监控流程，支持基于网页的历史数据查询，并促进课后教学反思。

专业报告导出：支持一键生成PDF格式的课堂绩效分析报告。

🛠️ 技术基础
算法层：Python 3.9+，MediaPipe（面部网格/姿态），OpenCV

接口层：

桌面：PyQt6，pyqtgraph（高性能实时绘图）

Web：Streamlit，Streamlit-WebRTC（异步流媒体处理）

数据层：SQLite（本地部署）/ PostgreSQL（云端）

样式库：Altair（现代可视化），Inter UI 字体
